I can't see any way of staging a device to this level in SOTI?

Johannes Harbs (harbs.johannes@gmail.com)
2020-04-22 13:57:53

*Thread Reply:* There is no such thing in SOTI like the WS1 Token enrollment or staging user. However, SOTI allows users to be added after the initial enrollment. So you could enroll a device without any user credentials and add the corresponding user afterwards. Downside to this: If you already push personal configurations (e.g. mail) right after enrollment, the profile is pushed without the specific user details and need to be re-pushed after assigning the user. Additionally, this requires additional admin effort. Sample scenario:

  1. Enroll with EnrollmentID without user authentication into a staging group
  2. assign the user to the device
